# MIDI Bridge — инструкция

## Что нужно

- Компьютер (Windows) с подключённой MIDI-клавиатурой
- iPad в той же Wi-Fi сети, что и компьютер
- Приложение ScorePractice открыто на iPad в Safari

---

## Шаг 1 — Запусти MIDI Bridge на компьютере

Открой папку `midi-bridge` в проводнике, дважды кликни на файл:

```
Запустить MIDI Bridge.bat
```

Откроется окно. Ты увидишь примерно:

```
Using MIDI input: "USB MIDI Keyboard" (index 0)
WebSocket server listening on ws://0.0.0.0:9001
Bridge running. Press Ctrl+C to stop.
```

**Окно должно оставаться открытым** пока играешь.

---

## Шаг 2 — Получи адрес для iPad

Приложение открыто через **GitHub Pages (https://)**, поэтому нужен ngrok — он создаёт безопасный туннель.

### Установить ngrok (один раз)

1. Зайди на [ngrok.com/download](https://ngrok.com/download), скачай для Windows
2. Распакуй `ngrok.exe` куда удобно (например, в папку `midi-bridge`)
3. Зарегистрируйся бесплатно на ngrok.com, скопируй свой токен
4. В cmd выполни один раз: `ngrok config add-authtoken ВАШ_ТОКЕН`

### Запустить туннель

В новом окне cmd:

```
ngrok http 9001
```

Ngrok покажет адрес вида:

```
Forwarding  https://xxxx-xx-xx.ngrok-free.app -> http://localhost:9001
```

Скопируй этот адрес (начинается с `https://`).

---

## Шаг 3 — Подключи iPad

На iPad открой приложение в Safari. В хедере нажми кнопку **🎹 MIDI**.

Откроется поле ввода. Введи адрес из ngrok, заменив `https://` на `wss://`:

```
wss://xxxx-xx-xx.ngrok-free.app
```

Нажми **Подключить**.

Статус изменится на **🎹 MIDI bridge** — соединение установлено.

---

## Альтернатива без ngrok (если открывать приложение локально)

Если хочешь открывать приложение **не через GitHub Pages**, а локально по сети:

1. В cmd в папке проекта: `npm run dev -- --host`
2. На iPad открой: `http://192.168.x.x:5173` (твой IP из `ipconfig`)
3. В поле MIDI введи: `ws://192.168.x.x:9001`

В этом случае ngrok не нужен.

---

## Шаг 4 — Играй

1. Открой партитуру (кнопка **+ Открыть партитуру** или **Тест**)
2. Нажми ноту на клавиатуре — курсор продвинется вперёд

---

## Возможные проблемы

| Проблема | Решение |
|---|---|
| Статус остаётся **🔌 нет связи** | Проверь что bridge запущен и ngrok тоже работает |
| Bridge пишет «No MIDI input devices found» | Подключи клавиатуру **до** запуска bridge |
| Клавиатура определилась не как [0] | Открой `.bat` в блокноте и добавь ` --port 1` в конец строки с `node` |
| ngrok показывает предупреждение при открытии | Нажми «Visit Site» — это нормально для бесплатного аккаунта |
